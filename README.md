# 🌦️ Weather Station

A responsive Weather Station web application built using **HTML, CSS, and JavaScript**. It uses the **Open-Meteo Forecast API** and **Open-Meteo Geocoding API** to display real-time weather conditions, hourly temperature trends, and a 7-day weather forecast.

## 🚀 Features

- 🌍 Search weather by city
- 🔍 City autocomplete suggestions
- 🌡️ Real-time weather information
- 📈 24-hour temperature trend
- 📅 7-day weather forecast
- 🌡️ Celsius/Fahrenheit unit toggle
- 🕒 Recently viewed cities
- 📱 Fully responsive design
- ⚡ Fast and lightweight
- 🔑 No API key required

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo Forecast API
- Open-Meteo Geocoding API
